# Guide de Contribution

Merci de l'intérêt que vous portez à **IgniteBill** ! Voici comment vous pouvez aider.

## Comment contribuer ?

### Signaler un bug
Utilisez le [template de Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) pour nous donner tous les détails nécessaires.

### Proposer une fonctionnalité
Utilisez le [template de Feature Request](.github/ISSUE_TEMPLATE/feature_request.md) pour nous expliquer votre idée.

### Envoyer une Pull Request (PR)
1. **Forkez** le projet.
2. Créez votre branche de fonctionnalité (`git checkout -b feature/AmazingFeature`).
3. Commitez vos changements (`git commit -m 'Add some AmazingFeature'`).
4. Pushez votre branche (`git push origin feature/AmazingFeature`).
5. Ouvrez une **Pull Request**.

## Standards de code
- Utilisez `flutter format` avant de commiter.
- Assurez-vous que `flutter analyze` ne remonte pas d'erreurs.
- Les tests unitaires doivent tous passer.

Merci encore pour votre aide !
