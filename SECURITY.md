# Politique de Sécurité

## Versions Supportées
Actuellement, seule la dernière version stable d'IgniteBill est supportée pour les correctifs de sécurité.

## Signaler une vulnérabilité
Si vous découvrez une faille de sécurité, merci de ne pas ouvrir d'issue publique. Envoyez plutôt un e-mail à l'adresse associée au profil du mainteneur ou utilisez le canal de contact privé du dépôt GitHub si disponible.

Nous ferons de notre mieux pour répondre et corriger le problème rapidement.
